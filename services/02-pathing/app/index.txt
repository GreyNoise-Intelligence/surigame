Exploiting Path Traversal Vulnerabilities

-----------------------------------------

Path traversal attacks involve manipulating URLs to access unauthorized files. Believe it or not, THIS VERY SITE is vulnerable!

All you have to do it access it by visiting: /../../../../../../../flag.txt

Go ahead! Use that URI!

(We also render directory listings, so feel free to browse the filesystem!)
